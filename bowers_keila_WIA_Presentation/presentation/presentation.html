What is DropzoneJS

DropzoneJS a library created by Matias Meno that allows you to upload files with drag'n'drop and via your document window.

What makes DropzoneJS Awesome?

  • Lightweight
    • Total Size: 88kb
    • Total Size Minified: 49kb
  • Indepedent but plays well with others
  • Works right out of the package
  • Amazing documentation
  • Easily and highly customizable

However, it doesn't handle server-side communications.

Make it work for you

DropzoneJS is set up to anticipate anything you could want to do.

It has options to customize style and file requirements.

You can use methods to remove, upload or call files.

Built in events.

Let's get started...

Setting up your JavaScript file


Dropzone.options.testDropzone = {
	//insert options, methods and events here
}

JavaScript with DropzoneJS




 Dropzone.options.myDropzone = {
        addRemoveLinks: true,
        acceptedFiles: "image/*",
        //acceptedFiles: "application/pdf",
        //acceptedFiles: ".gif",
        autoProcessQueue: false,

	init: function() {
	    var submit = document.getElementById("submit");
	    myDropzone = this;

	    submit.addEventListener("click", function() {
	      myDropzone.processQueue(); 
	    });

		myDropzone.on("complete", function(){ //do not use 'processing' with hang-up first item
			myDropzone.options.autoProcessQueue = true;
			myDropzone.processQueue();

		});

	}

};
